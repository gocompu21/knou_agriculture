## 제8장. 데이터 통신의 기초

### 8.1 통신의 개념

**통신**(communication)이란 정보를 전달하는 행위를 말한다. 광의의 통신은 서신, 봉화, 연날리기 등 모든 정보 전달 수단을 포함하지만, **협의의 통신**, 즉 **데이터 통신**은 **전기적, 전자적 매체**를 이용하여 정보를 교환하는 것을 의미한다. 대표적인 데이터 통신의 사례로는 **전보**와 **전화**가 있다. 서신이나 봉화, 연날리기 등은 전기적 매체를 사용하지 않으므로 데이터 통신에 해당하지 않는다.

**관련 문제**: (2018-20), (2019-21)

### 8.2 데이터 통신의 역사

컴퓨터와 통신을 결합시킨 역사에서 주요한 시스템들은 다음과 같다.

- **SAGE**: 컴퓨터와 통신을 결합시킨 최초의 컴퓨터 통신 시스템
- **SABRE**: 여객기의 좌석 예약을 위해 개발된 컴퓨터 통신망
- **ARPANET**: 미국 국방성에서 개발한 컴퓨터 통신망으로 인터넷의 전신
- **ALOHA**: 하와이대학에서 실험적으로 개발한 **무선 패킷교환 방식**의 컴퓨터 통신망

**관련 문제**: (2015-18)

### 8.3 데이터 통신 시스템의 구성요소

데이터 통신 시스템은 데이터를 보내는 **단말장치**(터미널), 데이터를 전달하는 **통신 채널**(전송 매체), 그리고 신호를 변환하는 **신호변환장치**(DCE)로 구성된다.

#### 8.3.1 모뎀

**모뎀**(modem)은 아날로그 회선을 이용하는 경우에 사용되는 대표적인 신호변환장치이다. 디지털 신호를 아날로그 신호로 변환하는 과정을 **변조**(modulation), 아날로그 신호를 디지털 신호로 변환하는 과정을 **복조**(demodulation)라 하며, 모뎀이라는 이름은 이 두 기능의 약어에서 유래하였다. 한편 디지털 회선을 이용하는 경우에는 **DSU**(Digital Service Unit)가 사용된다.

**관련 문제**: (2017-20), (2018-24), (2019-23)

### 8.4 전송 매체

데이터를 전송하는 물리적 경로를 **전송 매체**라 한다. 유선 매체와 무선 매체로 구분되며, 주요 유선 전송 매체는 다음과 같다.

- **꼬임선 케이블**(Twisted Pair Cable): 구리선 두 가닥을 꼬아 만든 전송 매체로, 전화선이나 LAN에 사용된다.
- **동축 케이블**(Coaxial Cable): 중심 도체를 절연체와 외부 도체로 감싼 구조로, 케이블 TV 등에 사용된다.
- **광섬유**(Optical Fiber): 유리나 플라스틱으로 된 가느다란 섬유를 이용하여 데이터를 **빛의 형태**로 전송하는 매체이다. 전자기 간섭의 영향을 거의 받지 않으며, 대용량 데이터를 고속으로 장거리 전송할 수 있다.

시험에서 "데이터를 빛으로 바꾸어 전송할 때 사용되는 전송 매체"를 묻는 문제가 매우 반복적으로 출제되므로, **광섬유**를 확실히 기억해야 한다.

**관련 문제**: (2016-26), (2017-24), (2019-28)

### 8.5 통신 프로토콜

**통신 프로토콜**(communication protocol)이란 통신을 원하는 두 개체 간에 **무엇을, 언제, 어떻게** 통신할 것인지 서로 약속한 규약을 의미한다. 프로토콜은 데이터의 형식, 전송 시점, 전송 방식 등을 정의하여 원활한 정보 교환을 가능하게 한다. TCP/IP, HTTP, FTP 등이 대표적인 통신 프로토콜이다.

**관련 문제**: (2016-27)

### 8.6 전송 방향

데이터의 전송 방향에 따라 통신 방식을 세 가지로 분류할 수 있다.

#### 8.6.1 단방향(Simplex) 전송

**단방향 전송**은 데이터가 한쪽 방향으로만 전송되는 방식이다. 수신자는 송신자에게 정보를 보낼 수 없다. **라디오 방송**이나 **텔레비전 방송**이 단방향 전송의 전형적인 사례이다.

**관련 문제**: (2019-29)

#### 8.6.2 반이중(Half-Duplex) 전송

**반이중 전송**은 데이터를 양방향으로 전송할 수 있지만, **동시에** 양방향으로 전송할 수는 없는 방식이다. 한 방향으로의 통신이 끝난 뒤 반대 방향으로의 통신이 이루어져 **정보 전송 방향이 교대로 바뀐다**. 무전기가 반이중 전송의 대표적인 사례이다.

**관련 문제**: (2017-25)

#### 8.6.3 전이중(Full-Duplex) 전송

**전이중 전송**은 데이터를 양방향으로 동시에 전송할 수 있는 방식이다. **전화망**이 전이중 전송의 전형적인 사례이다.

### 8.7 직렬 전송과 병렬 전송

데이터 전송 방식은 비트를 보내는 방법에 따라 **직렬 전송**과 **병렬 전송**으로 구분된다.

- **직렬 전송**(serial transmission): 하나의 통신 회선을 통하여 한 번에 **한 비트씩** 순서대로 전송하는 방식이다. 장거리 전송에 주로 사용된다.
- **병렬 전송**(parallel transmission): 여러 개의 전송로를 통하여 **동시에 여러 비트**를 전송하는 방식이다. 직렬 전송에 비해 빠르지만, 전송 거리가 짧을 때 주로 사용된다.

**관련 문제**: (2017-26), (2018-26)

### 핵심 키워드 요약

| 키워드 | 핵심 포인트 |
|--------|------------|
| 데이터 통신 | 전기적/전자적 매체를 이용한 정보 교환 (전보, 전화) |
| ALOHA | 하와이대학, 무선 패킷교환 방식 |
| 모뎀 | 아날로그-디지털 신호 상호 변환 (변조+복조) |
| 광섬유 | 빛으로 데이터 전송, 고속/장거리, 전자기 간섭 무관 |
| 통신 프로토콜 | 무엇을/언제/어떻게 통신할지의 약속된 규약 |
| 단방향 전송 | 한 방향만, 라디오/TV |
| 반이중 전송 | 양방향이나 교대로, 무전기 |
| 직렬 vs 병렬 | 한 비트씩 순서대로 vs 여러 비트 동시 전송 |
