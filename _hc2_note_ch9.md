## 제9장. 화훼의 번식

### 9.1 종자번식

화훼작물의 종자는 발아 시 빛에 대한 반응에 따라 **호광성(명발아)종자**와 **혐광성(암발아)종자**로 나뉜다.

#### 9.1.1 암발아종자(혐광성종자)

**암발아종자**는 빛이 없는 어두운 조건에서 발아율이 높아지는 종자로, 파종 시 충분한 복토가 필요하다. 대표적인 암발아종자를 가진 화훼로는 **시클라멘**, **백일홍**, **팬지** 등이 있다.

다만 혐광성 종자라는 용어는 빛을 싫어하여 어두운 곳에서 발아가 잘 된다는 뜻으로, 팬지를 혐광성 종자로 분류하는 문제도 출제되었다.

**관련 문제**: (2014-24), (2015-30), (2017-21), (2019-29), (2022-19)

#### 9.1.2 호광성종자(명발아종자)

**호광성종자**는 빛에 노출되어야 발아가 잘 되는 종자로, **금어초**, **칼랑코에**, **프리뮬러**, **디기탈리스**, **글록시니아** 등이 해당한다.

### 9.2 영양번식

#### 9.2.1 구근의 종류

화훼의 영양번식 기관은 종류에 따라 다양한 형태를 가진다.

| 번식기관 | 설명 | 해당 작물 |
|----------|------|-----------|
| **인경(비늘줄기, bulb)** | 비늘 모양 잎이 겹겹이 싸인 구조 | **나리**, **튤립** |
| **구경(알줄기, corm)** | 줄기가 비대해진 덩어리 | **글라디올러스**, **프리지어** |
| **괴경(덩이줄기, tuber)** | 줄기 일부가 비대하여 양분 저장 | **시클라멘** |
| **괴근(덩이뿌리, tuberous root)** | 뿌리가 비대해진 것 | **다알리아** |
| **근경(뿌리줄기, rhizome)** | 땅속에서 옆으로 뻗는 줄기 | **칸나** |

프리지어를 괴경이나 근경으로 연결하는 것은 잘못된 것이며, 올바른 연결은 **구경**이다. 튤립을 구경으로 연결하는 것도 잘못이며, 올바른 연결은 **인경**이다.

**관련 문제**: (2013-24), (2014-22), (2015-33), (2016-18), (2017-35)

#### 9.2.2 추식구근과 춘식구근

**추식구근**(9~10월에 심는 구근): 튤립, 프리지어, 나리, 수선화, 히아신스
**춘식구근**(봄에 심는 구근): 글라디올러스, 다알리아, 칸나, 아마릴리스

**관련 문제**: (2019-28)

#### 9.2.3 엽삽(잎꽂이)

엽삽이 **잘 되는** 화훼: **산세베리아**, 베고니아, 아프리칸 바이올렛
엽삽이 **잘 되지 않는** 화훼: **칼랑코에** (줄기 꺾꽂이가 더 효율적)

**관련 문제**: (2014-25), (2016-5)

### 9.3 접목번식

#### 9.3.1 접목선인장

우리나라의 접목선인장 생산과 육종은 세계적 수준이며, 국내 육성품종이 국내 재배면적의 **100%**를 차지한다. 접목선인장 생산에서 가장 많이 사용되는 접목 방법은 **안장접(안접, saddle graft)**이다. 안장접은 대목과 접수의 접합면을 평평하게 잘라 서로 맞닿게 하여 접합면적을 넓히는 방법으로, 다육질인 선인장에 특히 효과적이다.

**관련 문제**: (2013-22), (2013-35), (2014-26), (2017-25), (2018-30), (2019-30)

#### 9.3.2 장미의 접목

장미 번식에서 가장 많이 이용되는 접목법은 **합접(splice grafting)**이다. 대목과 접수의 굵기가 비슷할 때 사선으로 절단한 두 면을 맞대어 결합하는 방식이다.

**관련 문제**: (2015-31)

### 9.4 절화용 화훼

주로 **절화**로 이용되는 구근 화훼로는 **프리지어**와 **글라디올러스**가 대표적이다. 드라세나, 필로덴드론, 스킨답서스 등은 관엽식물로 분류된다.

**관련 문제**: (2015-24), (2017-22)

### 핵심 키워드 요약

| 키워드 | 핵심 포인트 |
|--------|------------|
| 암발아종자 | 시클라멘, 백일홍, 팬지 — 어둡게 복토 |
| 인경(bulb) | 나리, 튤립 |
| 구경(corm) | 글라디올러스, 프리지어 |
| 괴경(tuber) | 시클라멘 |
| 괴근 | 다알리아 |
| 근경(rhizome) | 칸나 |
| 추식구근 | 튤립, 프리지어, 나리 (9~10월) |
| 춘식구근 | 글라디올러스, 다알리아 (봄) |
| 엽삽 잘 됨 | 산세베리아, 베고니아 |
| 접목선인장 | 안장접(안접), 국내 육성 100% |
| 장미 접목 | 합접(splice grafting) |
