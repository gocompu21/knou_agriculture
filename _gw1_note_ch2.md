## 제2장. 띄어쓰기와 외래어 표기

### 2.1 외래어 표기법의 원칙

외래어 표기법 제3장 제7항에 따르면, 받침에는 **ㄱ, ㄴ, ㄹ, ㅁ, ㅂ, ㅅ, ㅇ** 7개 자음만 사용하도록 규정하고 있다. 따라서 'cake'의 종성 발음 [k]는 받침 'ㄱ'으로만 표기해야 하므로, '케잌'이 아니라 **케이크**가 올바른 표기이다.

기출에 반복 출제되는 외래어의 올바른 표기는 다음과 같다.

- **케이크** (O) / 케잌, 케익 (X): cake
- **주스** (O) / 쥬스 (X): juice (외래어 표기법에서 'ㅠ' 대신 'ㅜ' 사용)
- **로봇** (O) / 로보트 (X): robot
- **디스켓** (O) / 디스켙 (X): diskette
- **텔레비전** (O) / 텔레비젼, 테레비 (X): television
- **그리스** (O) / 그리이스 (X): Greece

외래어 표기에서 특히 주의할 점은 받침에 'ㅋ', 'ㅌ' 등을 사용하지 않는다는 것과, 'ㅈ' 뒤에 'ㅠ'를 쓰지 않는다는 점이다.

**관련 문제**: (2013-5), (2014-7), (2015-5), (2016-4), (2017-5), (2018-4), (2019-6), (2024-5)

### 2.2 의존 명사와 접두사의 띄어쓰기

의존 명사는 홀로 쓰이지 못하고 반드시 앞말에 기대어 쓰이지만, **앞말과 띄어 쓰는 것이 원칙**이다. 반면 접두사는 뒷말에 **붙여 쓴다**.

#### 2.2.1 '수'의 띄어쓰기

**수**(數)가 '셈' 또는 '수효'를 뜻하는 **의존 명사**일 때는 앞말과 띄어 쓴다. '사람 수가 모자란다', '가구별 평균 자녀 수'에서 '수'는 의존 명사이므로 띄어 쓴다. 반면 '수년', '수십'에서 **수-**는 '몇몇의' 또는 '여럿의' 의미를 더하는 **접두사**이므로 뒷말에 붙여 쓴다. '수 많은'은 잘못된 표기이며, **수많은**이 올바르다.

#### 2.2.2 수 관형사와 단위 의존 명사

수를 나타내는 **관형사**는 뒤에 오는 단위 의존 명사와 띄어 쓴다. '수 백 미터'에서 '수'는 관형사, '백'과 '미터'는 의존 명사이므로 각각 띄어 쓴다. 또한 '천 원', '오만 원'처럼 화폐 단위도 수 관형사와 띄어 쓴다. 다만 **-짜리**는 접미사이므로 앞말에 붙여 쓴다: '천 원짜리', '오백 원짜리'.

**-어치**는 의존 명사이므로 앞말과 띄어 쓰는 것이 원칙이다. '만원 어치', '오천 원 어치'가 올바른 띄어쓰기이다.

'수십 년간'에서 '수십'은 합성어로 붙여 쓰고, '간(間)'은 기간을 나타내는 접미사로 앞말에 붙여 쓴다. 따라서 '수 십년 간'은 잘못된 표기이며, **수십 년간**이 올바르다.

**관련 문제**: (2013-6), (2014-5), (2014-6), (2015-3), (2015-4), (2016-3), (2017-3), (2019-5), (2024-7)

### 2.3 보조사와 조사의 띄어쓰기

**조사**는 앞말에 붙여 쓴다. 접속 조사 '과'는 '우리말과 우리글'처럼 앞말에 붙이고, 보조사 '-만'도 '열심히만'처럼 붙여 쓴다.

연결 어미 **-든(지)**는 앞의 내용과 상관없이 뒤의 내용이 이어짐을 나타내는 어미로, 앞말에 붙여 쓴다. '공부를 하든지 운동을 하든지 간에'에서 '-든지'는 어미이므로 앞말에 붙이고, **간에**는 의존 명사 '간'에 조사 '에'가 붙은 형태이므로 앞말과 띄어 쓴다.

**관련 문제**: (2013-7), (2017-3), (2019-5)

### 2.4 '-당하다'의 띄어쓰기

명사 뒤에 붙어 피동의 의미를 더하는 **-당하다**는 앞말에 **붙여 쓴다**. '거절당하다'가 올바른 표기이며, '거절 당하다'는 잘못된 띄어쓰기이다. 이는 '-당하다'가 접미사적 성격을 가지기 때문이다.

**관련 문제**: (2015-4), (2016-3), (2017-4)

### 2.5 기타 자주 출제되는 띄어쓰기

- **상관없다** (O): '상관(이) 없다'는 붙여 쓸 수도 있다
- **은연중에** (O) / 은연 중에 (X): '은연중'은 하나의 명사
- **우리말** (O) / 우리 말 (X): 하나의 합성어
- **자리잡다** (O): 합성 동사
- **한쪽** (O) / 한 쪽 (X): 하나의 명사

**관련 문제**: (2013-7), (2016-3), (2017-4)

### 핵심 키워드 요약

| 키워드 | 핵심 포인트 |
|--------|------------|
| 외래어 받침 | ㄱ, ㄴ, ㄹ, ㅁ, ㅂ, ㅅ, ㅇ 7개만 허용 |
| 케이크/주스/로봇/디스켓 | 반복 출제되는 올바른 외래어 표기 |
| 의존 명사 | 앞말과 띄어 쓰기 (수, 어치, 간 등) |
| 접두사 수- | 뒷말에 붙여 쓰기 (수많은, 수년) |
| -짜리 | 접미사로 앞말에 붙여 쓰기 (천 원짜리) |
| -당하다 | 접미사적 성격, 앞말에 붙여 쓰기 (거절당하다) |
| -든지 | 연결 어미, 앞말에 붙여 쓰기 |
