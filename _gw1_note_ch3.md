## 제3장. 어휘 사용

### 3.1 유사 어휘의 올바른 사용

한국어에서 의미가 유사하지만 쓰임이 다른 어휘를 혼동하는 사례가 자주 출제된다.

**'다르다'와 '틀리다'**는 구별하여 사용해야 한다. **다르다**는 비교 대상 사이에 차이가 있음을 나타내고, **틀리다**는 사실이나 기준에 맞지 않음을 나타낸다. '너하고 나는 성격이 틀려'는 잘못된 표현이며, '너하고 나는 성격이 **달라**'가 올바르다.

**'적다'와 '작다'**도 혼동되기 쉽다. **적다**는 수량이나 정도가 기준에 미치지 못할 때, **작다**는 크기나 부피를 나타낼 때 사용한다. '몸집이 적은'은 잘못된 표현이며, '몸집이 **작은**'이 올바르다.

**'내성적'의 반대말**은 **외향적**이다. '외성적'이라는 단어는 존재하지 않는 잘못된 표현이므로, '내성적인 데 반해 외성적이다'가 아니라 '내성적인 데 반해 **외향적**이다'로 써야 한다.

**관련 문제**: (2013-8), (2014-8), (2015-6), (2017-6)

### 3.2 관용 표현의 올바른 사용

관용어나 연어 관계를 잘못 사용하는 경우가 빈번하게 출제된다.

**사달이 나다**는 '좋지 않은 일이나 사고가 생기다'라는 의미의 올바른 관용구이다. '사단이 나다'는 어색한 표현으로, **사달**이 올바른 어휘이다.

**치켜세우다**와 **추켜올리다**는 혼동되기 쉽다. 남을 높여 칭찬하는 뜻으로는 **추켜세우다** 또는 **치켜세우다**를 사용해야 하며, '추켜세우다'가 더 적절한 경우가 많다.

**염치 불고하고**가 올바른 관용 표현이다. '불고(不顧)하다'는 '돌보지 아니하다'의 뜻으로 '체면을 돌보지 않고'라는 의미이다. '염치 **불구**하고'는 잘못된 표현이다.

**관련 문제**: (2013-9), (2014-9), (2015-7), (2016-5), (2017-7), (2018-5), (2019-7)

### 3.3 성적·수준 관련 어휘의 호응

**성적**의 변화를 표현할 때 '상승하다'보다 **향상되다**가 더 적절하다. '상승하다'는 주로 수치나 양이 올라가는 것(물가 상승, 기온 상승)에 사용하고, 수준이나 품질이 나아지는 것은 **향상**이라 표현한다.

**동점**과 **동률**의 구별도 중요하다. 점수가 같을 때는 **동점**을 이루었다고 하며, '동률'은 비율이 같을 때 사용한다. '2:2 동률'이 아니라 '2:2 **동점**'이 올바르다.

**관련 문제**: (2013-8), (2014-8), (2016-5), (2016-6), (2017-6)

### 3.4 '와중'과 '탓/덕분'의 올바른 쓰임

**와중(渦中)**은 '소용돌이의 한가운데'라는 뜻으로, 복잡하고 혼란스러운 상황을 비유적으로 이르는 말이다. '전란의 와중에'와 같이 부정적이거나 혼란스러운 맥락에서 사용하는 것이 자연스럽다.

**탓**은 주로 부정적인 원인을 나타내고, **덕분**은 긍정적인 원인을 나타낸다. '꾸준히 노력한 탓에 성공했다'는 잘못된 표현이며, '꾸준히 노력한 **덕분에** 성공했다'가 올바르다.

**관련 문제**: (2014-9), (2016-6), (2017-7)

### 3.5 기타 자주 출제되는 어휘

- **구설수에 오르다**: '남의 구설수에 오르다'는 '남의'가 중복되어 어색하다
- **재원(才媛)**: 재주 있는 여성을 이르는 말로, 남성에게 사용하면 부적절
- **서거(逝去)**: 이미 돌아가신 분에게 '영면을 기원합니다'는 어색한 표현
- **알은척하다** (O): '아는 체하다'의 올바른 표현
- **제치다** (O) / 제끼다 (X): '건너뛰다'의 표준 표현

**관련 문제**: (2013-9), (2019-7), (2019-8), (2019-9), (2024-6)

### 핵심 키워드 요약

| 키워드 | 핵심 포인트 |
|--------|------------|
| 다르다 / 틀리다 | 차이 vs 오류 |
| 적다 / 작다 | 수량·정도 vs 크기·부피 |
| 외향적 | '내성적'의 올바른 반대말 ('외성적' X) |
| 사달이 나다 | '사단이 나다' (X) |
| 염치 불고하고 | '불구하고' (X), '불고(不顧)' = 돌보지 않다 |
| 향상되다 / 상승하다 | 수준·품질 vs 수치·양 |
| 동점 / 동률 | 점수 vs 비율 |
| 탓 / 덕분 | 부정적 원인 vs 긍정적 원인 |
