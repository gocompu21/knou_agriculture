## 제11장. 유기가공식품과 표시제도

### 11.1 유기농산물 함량에 따른 표시기준

유기가공식품의 표시는 최종제품에 남아 있는 유기농산물 원재료의 함량 비율에 따라 차등 적용된다.

| 유기농산물 함량 | 주표시면 표시 | 다른 표시면 표시 |
|----------------|-------------|----------------|
| 원재료의 100% | 가능 | 가능 |
| 95% 이상 | 가능 | 가능 |
| **70% 이상 95% 미만** | **불가** | **'유기' 용어 사용 가능** |
| 70% 미만 | 불가 | 불가 |

최종제품에 남아 있는 원재료의 **70% 이상 95% 미만**이 유기농산물인 경우, 식품의 **주표시면에는 '유기'라는 용어를 표시할 수 없으나**, 다른 표시면에는 '유기농' 또는 '유기농산물'이라는 용어를 사용할 수 있다. 이 경우 유기농산물 함량비율과 '유기농 원료' 문구를 함께 표시해야 한다. 이는 95% 이상인 제품과 차등을 두면서도 소비자가 제품의 유기농 함량을 명확히 인지할 수 있도록 하기 위한 기준이다.

**관련 문제**: (2013-33)

### 11.2 유기가공식품의 표시 가능 조건

국내에서 생산된 식품에 유기가공식품 표시를 할 수 있는 경우는, **유기농산물을 원재료로 하며 유기가공식품에 사용하는 용기ㆍ포장은 재활용이 가능하거나 생물분해성 재질**인 경우이다. 유기식품은 생산부터 소비까지 전 과정에서 환경친화적인 접근을 추구하며, 친환경적인 포장재 사용은 이러한 지속가능한 가치와 부합한다.

반면 다음의 경우에는 유기가공식품 표시를 할 수 **없다**.

- 동일 원재료에 대하여 유기농산물과 **비유기농산물을 혼합**하여 사용하는 경우
- **방사선 조사 처리**된 원재료를 사용하는 경우
- **유전자변형 농산물** 또는 식품첨가물을 사용한 원재료인 경우

**관련 문제**: (2013-34)

### 11.3 유기가공식품의 제조ㆍ가공 방법

#### 11.3.1 기본 원칙

유기가공식품의 제조ㆍ가공은 유기원료의 유기적 특성을 최대한 유지하기 위해 최소한의 가공 방법을 사용하는 것을 원칙으로 한다. **기계적ㆍ물리적 또는 생물학적**으로 제조ㆍ가공하며, 허용된 식품첨가물이라도 **최소량만을 사용**하여야 한다.

여기서 '식품첨가물은 사용하지 아니하여야 한다'는 설명은 옳지 않다. 유기가공식품 제조 시에도 유기가공식품 기준에 적합하게 제조된 경우 **승인된 일부 식품첨가물은 사용할 수 있으며**, 다만 최소량만을 사용해야 한다는 것이 정확한 기준이다.

**관련 문제**: (2013-35), (2016-34)

#### 11.3.2 제조ㆍ가공 세부 기준

유기가공식품 표시를 위한 제조ㆍ가공 방법의 세부 기준은 다음과 같다.

- 유기가공식품과 비유기가공식품을 **동일한 시간에 동일한 설비로 제조ㆍ가공하여서는 안 된다.**
- 비유기가공식품을 제조ㆍ가공한 설비라도 **이물질을 완전히 제거하고 세척을 철저히 하면** 유기가공식품 제조ㆍ가공 설비로 사용할 수 있다.
- 유기가공식품과 원료유기농산물은 비유기가공식품 및 비유기원료농산물과 **따로 보관ㆍ저장**하여야 한다.
- 식품과 직접 접촉하는 부분의 살균은 화학약품 대신 적절한 방법으로 소독해야 한다.

**관련 문제**: (2016-34)

### 11.4 유기가공식품의 원재료 기준

유기가공식품 또는 이와 유사한 용어를 표시하기 위한 원재료 기준은 다음과 같다.

- 유기농산물 인증을 받은 농ㆍ축ㆍ임산물이 **원재료의 95% 이상**이어야 한다.
- 동일한 원재료는 유기농산물과 **비유기농산물의 혼합사용이 불가**하다.
- **유전자재조합 식품** 또는 식품첨가물은 사용불가하다.
- 식품위생을 위한 원재료의 **방사선 조사 처리는 금지**된다. 방사선 조사는 식품의 본래 특성을 변경하고 유기농업의 원칙에 부합하지 않기 때문이다.

**관련 문제**: (2014-35)

### 11.5 유기가공식품의 도형 표시

유기가공식품의 도형 표시에 관한 규정은 다음과 같다.

표시 도형 내부의 **'유기식품'과 'ORGANIC' 글자 색상은 표시 도형 색상과 동일하게** 한다. 이는 표시의 통일성을 유지하기 위한 규정이다.

한편 다음 설명들은 올바르지 않다.

- '표시 도형의 국문 및 영문 글자의 활자체는 궁서체로 한다' → 궁서체로 한정하지 않는다.
- '표시 도형의 크기는 포장재의 크기에 관계없이 지정된 크기로 한다' → 포장재의 크기에 따라 적절히 조절할 수 있다.
- '표시 도형의 색상은 백색을 기본 색상으로 한다' → 녹색 또는 파란색이 기본 색상이다.

**관련 문제**: (2014-34), (2017-34)

### 핵심 키워드 요약

| 키워드 | 핵심 포인트 |
|--------|------------|
| 유기농산물 함량 70~95% | 주표시면 '유기' 불가, 다른 표시면에는 사용 가능 |
| 유기가공식품 표시 조건 | 유기 원재료 + 재활용/생물분해성 포장재 |
| 방사선 조사 처리 | 유기가공식품에서 엄격히 금지 |
| 유전자변형 원재료 | 유기가공식품에서 사용 불가 |
| 제조ㆍ가공 방법 | 기계적, 물리적, 생물학적 방법 / 허용 첨가물 최소량 사용 가능 |
| 동일 설비 사용 | 동일 시간 동일 설비로 유기/비유기 동시 제조 금지 |
| 원재료 95% 이상 | 유기가공식품 표시를 위한 유기 원재료 함량 기준 |
| 동일 원재료 혼합 | 유기+비유기 혼합 사용 불가 |
| 도형 표시 색상 | 내부 글자 색상은 도형 색상과 동일 |
