## 제3장. 자기디스크와 SSD

### 3.1 하드디스크(HDD)

**하드디스크**(Hard Disk Drive, HDD)는 자성 물질로 코팅된 원형의 **플래터**(platter)를 고속으로 회전시키면서, **자기 헤드**를 이용하여 데이터를 읽고 쓰는 대표적인 보조기억장치이다. 하드디스크는 밀폐된 구조로 제작되어 내부의 먼지나 이물질로 인한 손상을 방지하며, 사용자가 디스크를 개별적으로 교체할 수 없다.

하드디스크의 성능에 가장 큰 영향을 미치는 요소는 **디스크의 회전 속도**(RPM)이다. 플래터의 회전 속도가 빠를수록 데이터 접근 시간이 단축되고 데이터 전송률이 높아져 전체적인 성능이 향상된다. 하드디스크는 **직접접근 저장장치**로서 원하는 데이터의 위치에 바로 접근할 수 있으며, 읽기와 쓰기가 모두 가능하다. 또한 내부에 모터와 같은 기계적 장치가 있어 진동이나 충격에 취약할 수 있다는 특징이 있다.

**관련 문제**: (2015-3), (2019-4)

### 3.2 SSD(Solid State Drive)

**SSD**(Solid State Drive)는 **플래시 메모리**(NAND 플래시)를 사용하여 데이터를 저장하는 **반도체 기반** 보조기억장치이다. 하드디스크와 달리 모터나 회전하는 플래터와 같은 움직이는 기계 부품이 전혀 없으므로 여러 면에서 뚜렷한 장점을 가진다.

- **가볍고 휴대가 용이**하다.
- **전력 소모가 적어** 배터리 수명이 연장된다.
- **읽기/쓰기 속도가 매우 빠르다**.
- **충격에 강하고 내구성이 뛰어나다**.
- **소음이 없다**.

이러한 장점 때문에 SSD는 하드디스크를 대체하여 컴퓨터의 주 저장 장치로 사용될 수 있으며, 특히 노트북 컴퓨터에서 가벼움, 빠른 프로그램 실행, 긴 배터리 사용 시간을 모두 원할 때 가장 적합한 보조기억장치이다. 다만 SSD는 기록 및 삭제 횟수에 제한이 있어 반영구적이지는 않으며, 대용량 구성 시 하드디스크에 비해 비용이 높다는 단점도 있다.

**관련 문제**: (2015-5), (2016-3), (2017-5), (2018-3), (2019-5)

### 3.3 USB 플래시 드라이브

**USB 플래시 드라이브**는 USB 포트에 꽂아 사용하는 소형 휴대용 저장장치이다. SSD와 마찬가지로 플래시 메모리를 사용하며, **핫 플러깅**(Hot Plugging)을 지원하여 컴퓨터 전원이 켜진 상태에서도 자유롭게 연결하거나 분리할 수 있다. 다만 USB 플래시 드라이브를 통한 데이터 유출이나 바이러스 감염의 위험이 있으므로 보안에 주의가 필요하다.

**관련 문제**: (2018-5)

### 3.4 디스크 어레이(Disk Array)

**디스크 어레이**는 여러 개의 독립적인 디스크 드라이브를 하나로 묶어 논리적인 단위로 사용하는 기술이다. 대표적으로 **RAID**(Redundant Array of Independent Disks) 기술이 있으며, 디스크 어레이를 활용하는 주된 목적은 **데이터 저장의 신뢰성이나 성능을 높이는 것**이다. 예를 들어 RAID 1은 데이터를 미러링하여 신뢰성을 높이고, RAID 0은 데이터를 분산 저장하여 입출력 성능을 향상시킨다. 비용 절감이나 휴대성 향상이 아닌, 신뢰성과 성능이 핵심 목적이라는 점에 유의해야 한다.

**관련 문제**: (2017-3)

### 3.5 광 디스크

**광 디스크**는 **레이저**를 이용하여 데이터를 읽고 쓰는 보조기억장치이다. 기록 방식에 따라 크게 세 가지로 분류된다.

#### 3.5.1 읽기 전용(ROM) 디스크

공장에서 데이터가 기록된 상태로 제작되며, 사용자가 데이터를 수정하거나 추가로 기록할 수 없다. **CD-ROM**이 대표적이다. CD는 원래 74분의 스테레오 음악을 저장할 수 있도록 개발되었다.

#### 3.5.2 WORM(Write Once Read Many) 디스크

**WORM 형식**의 광 디스크는 공백 상태로 제작된 디스크에 **1회에 한해** 데이터를 기록할 수 있으며, 일단 기록된 데이터는 삭제하거나 변경할 수 없고 여러 번 읽을 수만 있다. **CD-R**, **DVD-R**, **BD-R** 등이 이에 해당한다.

**관련 문제**: (2017-4), (2019-3)

#### 3.5.3 재기록 가능(Rewritable) 디스크

자기디스크처럼 데이터를 여러 번 쓰고 지울 수 있는 광 디스크이다. **CD-RW**, **DVD-RW** 등이 해당하며, 약 1,000회 정도 재기록이 가능하다.

### 3.6 광 디스크의 저장 용량 비교

광 디스크는 종류에 따라 저장 용량에 큰 차이가 있다.

- **CD**: 약 700MB
- **DVD**: 단일층 약 4.7GB, 이중층 약 8.5GB
- **블루레이(Blu-ray) 디스크**: 단일층 약 25GB, 이중층 약 50GB

저장 용량이 가장 큰 광 디스크는 **블루레이 디스크**이며, **고선명(HD) 비디오** 저장에 적합한 디지털 기록매체이다. DVD 역시 효율적인 압축 기술을 사용하면 2시간 분량의 HD 비디오 저장이 가능하다. 참고로 플로피디스크는 자기디스크의 일종으로 광디스크에 해당하지 않는다.

**관련 문제**: (2015-4), (2016-4), (2018-4)

### 핵심 키워드 요약

| 키워드 | 핵심 포인트 |
|--------|------------|
| 하드디스크(HDD) | 자기 플래터 회전, 회전 속도가 성능 좌우, 직접접근 저장장치 |
| SSD | 플래시 메모리 기반, 기계 부품 없음, 가볍고 빠르고 저전력 |
| USB 플래시 드라이브 | 소형 휴대용, 핫 플러깅 지원, 보안 주의 필요 |
| 디스크 어레이(RAID) | 복수 디스크 묶음, 신뢰성/성능 향상 목적 |
| WORM | 1회 기록, 다회 읽기 (CD-R, DVD-R, BD-R) |
| 블루레이(Blu-ray) | 최대 용량 광디스크 (25/50GB), HD 비디오 저장에 적합 |
