## 제1장. 맞춤법과 표기법

### 1.1 연결 어미의 올바른 표기

한국어에서 특정 연결 어미는 발음과 표기가 혼동되기 쉬워 시험에 자주 출제된다. 대표적으로 **-(으)ㄹ는지**는 막연한 의문이나 추측을 나타내는 연결 어미로, '될런지', '있을런지'와 같은 잘못된 표기를 주의해야 한다. 올바른 형태는 **될는지**, **있을는지**이다. '되다'의 어간 '되-'에 '-ㄹ는지'가 붙어 '될는지'가 되며, 이는 어떤 일의 실현 가능성에 대한 의문을 나타낸다.

또한 목적이나 의도를 나타내는 연결 어미 **-려고**는 동사 '하다'의 어간 '하-' 뒤에 결합하여 **하려고**가 된다. '할려고'는 잘못된 표기이다.

**-노라고**와 **-느라고**의 구별도 중요하다. '하노라고'는 말하는 이가 어떤 행동을 했음을 겸손하게 표현할 때 사용하며, '하느라고'와 의미가 다르다. 마찬가지로 **그러고 나서**가 순차적 행동을 나타내는 올바른 표현이며, '그리고 나서'는 바르지 않다.

**관련 문제**: (2013-1), (2014-1), (2017-1), (2018-1), (2019-1), (2019-2)

### 1.2 '-로서'와 '-로써'의 구별

**-로서**는 **자격이나 신분**을 나타낼 때 사용하고, **-로써**는 **수단이나 도구**를 나타낼 때 사용한다. 예를 들어, '친구로서 충고를 하다'에서는 자격을 나타내므로 '-로서'가 옳고, '일에 몰두함으로써 괴로움을 잊다'에서는 수단을 나타내므로 '-로써'가 옳다. 또한 '죽음으로써 나라를 지켰다'는 수단의 의미이므로 '-로써'가 올바른 표기이다.

**-므로**와 **-ㅁ으로**의 구별도 빈출 주제이다. **-(으)므로**는 이유나 원인을 나타내는 연결 어미이고, **-ㅁ으로(써)**는 명사형 어미 '-ㅁ' 뒤에 조사 '-으로(써)'가 붙어 수단을 나타낸다. '품행이 방정하므로 표창함'이 올바른 표현이며, '품행이 방정함으로'는 이유를 나타내는 문맥에서 잘못된 표기이다.

**관련 문제**: (2013-1), (2014-2), (2017-1), (2019-1)

### 1.3 종결 어미와 전성 어미의 표기

**-대**와 **-데**의 구별이 자주 출제된다. **-대**는 남의 말을 전달하는 간접 인용의 종결 어미이고, **-데**는 과거 경험을 회상하여 말하는 종결 어미이다. '영수가 그러는데 오늘은 학교에서 민희를 못 봤대'에서 '-대'는 전달의 의미로 올바른 표기이며, '신기하데'는 회상의 의미로 올바른 표기이다.

형용사 **'같다'**의 활용에서 '같애'는 잘못된 표기이며, 올바른 형태는 **같아**이다. 어간 '같-' 뒤에 어미 '-아'가 붙어 '같아'로 활용된다.

형용사의 관형사형 어미 적용도 중요하다. **걸맞다**는 형용사이므로 현재 시제의 관형사형은 **-(으)ㄴ**을 붙여 **걸맞은**으로 써야 한다. '걸맞는'은 잘못된 표기이다. 마찬가지로 '알맞다'의 바른 활용형은 **알맞은**이다.

**관련 문제**: (2013-1), (2013-2), (2013-3), (2014-4), (2015-2), (2016-2), (2018-2)

### 1.4 자주 틀리는 어휘의 올바른 표기

기출문제에 반복 출제되는 어휘 표기를 정리하면 다음과 같다.

- **왠지** (O) / 웬지 (X): '왜인지'의 줄임말로 '왜 그런지 모르게'의 의미
- **웬** (O): '어찌 된', '어떠한'의 의미 (예: 이게 웬 일이야)
- **부딪히다** (O): '부딪다'의 피동사
- **부딪치다** (O): '부딪다'의 강세사
- **김치찌개** (O) / 김치찌게 (X): '-개'가 올바른 접미사
- **덮개** (O) / 덮게 (X)
- **창피하다** (O) / 챙피하다 (X)
- **홀아비** (O)
- **잇달다** (O), **잇따르다** (O): 모두 표준어
- **어떡해** (O) / 어떻하지 (X): '어떻게 해'의 줄임
- **금세** (O) / 금새 (X): '금시에'가 줄어든 말
- **오랜만에** (O) / 오랫만에 (X), 오랫동안 (O)
- **요컨대** (O): '요약하여 말하면'의 의미
- **귓띔** (O) / 귀뜸 (X)
- **눈곱** (O) / 눈꼽 (X)
- **놀래다** (O): '놀라게 하다'의 사동사

**관련 문제**: (2013-2), (2013-3), (2013-4), (2014-3), (2014-4), (2015-1), (2015-2), (2016-1), (2017-2), (2019-3), (2019-4)

### 1.5 '수'의 품사에 따른 표기

**수**가 '셈'이나 '셀 수 있는 수효'를 뜻하는 **의존 명사**로 쓰일 때는 앞말과 띄어 쓴다(예: 사람 수, 가구별 평균 자녀 수). 반면 **수많은**에서 **수-**는 '많거나 큼'의 뜻을 더하는 **접두사**이므로 뒷말에 붙여 쓴다. 접두사는 독립적으로 쓰이지 못하므로 단어의 자격을 갖지 못한다.

**관련 문제**: (2018-3)

### 1.6 '-십시오'와 '-십시요'

'해 주십시오'가 올바른 표기이며, '해 주십시요'는 잘못된 표기이다. 또한 '아니오'가 맞으며, '아니요'는 감탄의 뜻일 때만 사용한다. '마십시오'가 올바른 종결이며, '마십시요'는 잘못이다.

**관련 문제**: (2013-4), (2015-1), (2024-4)

### 핵심 키워드 요약

| 키워드 | 핵심 포인트 |
|--------|------------|
| -(으)ㄹ는지 | 막연한 의문·추측의 연결 어미, '될는지'가 올바른 표기 |
| -로서 / -로써 | 자격·신분 vs 수단·도구 |
| -(으)므로 / -ㅁ으로 | 이유·원인의 연결 어미 vs 명사형+조사(수단) |
| -대 / -데 | 간접 인용(전달) vs 과거 경험(회상) |
| 걸맞은 / 알맞은 | 형용사의 관형사형은 -(으)ㄴ |
| 왠지 / 웬 | '왜인지' 줄임 vs '어찌 된' |
| 김치찌개, 덮개 | '-개'가 올바른 접미사 |
| 수(의존 명사) / 수-(접두사) | 띄어 쓰기 vs 붙여 쓰기 |
